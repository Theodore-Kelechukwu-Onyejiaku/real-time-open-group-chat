# Open Group Chat

## Description
A chat application that allows users connect, send messages, view number of connected or online users, know who is typing, disconnect and many more.

## Built using the following technologies
- [NodeJs](https://nodejs.org/en/)
- [ReactJs](https://reactjs.org/)
- [Socket.io](https://socket.io/)
- [Tailwind CSS](https://tailwindcss.com/)
